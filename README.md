## Getting Started

### Installation

To get started with You&Me Shop, follow these steps:

**1. Clone the project**

```bash

git clone https://github.com/Vuthy-Tourn/HW6-Small-Web.git

```

**2. Go to the project directory**

```bash

cd HW6-Small-Web

```

**3. Install dependencies**

```bash

npm install

```

**4. Start the server**

```bash

npm run parcel

```
